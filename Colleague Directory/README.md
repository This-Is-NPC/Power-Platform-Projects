# Colleague Directory

##### Para T.I um aplicativo de gerenciamento de ramais, para os usuários um aplicativo amigável para pesquisar os números de todos os colegas disponiveis na empresa

> Tecnologias usadas:
  * Power Apps canvas
  * Power Automate
  * Solutions
  * Power Platform Pipelines
  * Outlook 365
  * SharePoint
  * PowerShell
  * Teams
